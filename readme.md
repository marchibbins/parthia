## Parthia

A site built with [Jekyll](http://jekyllrb.com/).

1. Install Jekyll

        $ gem install jekyll

2. Serve

        $ cd parthia
        $ jekyll serve

3. Browse to [http://localhost:4000](http://localhost:4000)
